## How to Use

1. Browse the category folders to find the prompt you need
2. Open the markdown file and copy the prompt section
3. Paste into your preferred AI tool (Claude, ChatGPT, etc.)
4. Customize as needed for your specific use case

## Adding New Prompts

When adding new prompts:
1. Place them in the appropriate category folder
2. Use descriptive filenames (lowercase-with-hyphens.md)
3. Include a "Purpose" and "Usage" section at the top
4. Add version history at the bottom
5. Update this README with a link and description
6. Commit changes with descriptive message

## Version Control

This library uses git for version tracking. To save changes:

```bash
cd ~/Documents/prompts
git add .
git commit -m "Description of changes"
```

## Tips

- Keep prompts generic and reusable
- Document any variables or customization points
- Track what works well and iterate on prompts
- Consider creating variations for different use cases
